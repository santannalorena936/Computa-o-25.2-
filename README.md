# Manual Técnico – Robô Otto Wheels (Customizado)

## 1. Introdução

Este manual técnico apresenta o **Robô Otto Wheels**, descrevendo sua concepção, modelagem, organização do sistema e funcionamento geral. São abordadas as funcionalidades, especificações técnicas, componentes eletrônicos e o processo de instalação do software.

O robô foi projetado e montado virtualmente no software **Fusion 360**, no qual todo o processo de modelagem tridimensional, definição das peças, encaixes e estrutura mecânica foi realizado previamente à implementação física. A utilização do Fusion 360 proporcionou maior precisão no desenvolvimento, facilitando a visualização do sistema, a análise da montagem e a integração entre os componentes mecânicos e eletrônicos.

---

## 2. Visão Geral do Sistema

O **Robô Otto Wheels** é composto por subsistemas mecânicos, eletrônicos e de controle, integrados para a execução de tarefas automatizadas, com foco em aplicações educacionais e simulações de processos utilizados em canteiros de obras.

### 2.1 Principais Funcionalidades

- Movimentação controlada de atuadores  
- Manipulação de objetos por meio da garra acoplada ao braço robótico  
- Detecção e desvio de obstáculos  
- Operação em modo manual e/ou automático  
- Compactação de solo por meio do rolo compressor, simulando processos reais da construção civil

## 4. Componentes Eletrônicos

Os principais componentes eletrônicos utilizados no **Robô Otto Wheels** são:

- Microcontroladores  
- Servomotores  
- Sensores (ultrassônico e infravermelho)  
- Fonte de alimentação  
- Cabos e conexões  

---

## 5. Instalação do Código-Fonte

### 5.1 Requisitos

- IDE de programação compatível (ex.: Arduino IDE)  
- Bibliotecas necessárias para o funcionamento correto do robô  
- Cabo USB para comunicação com o microcontrolador  

### 5.2 Procedimento de Instalação

1. Conectar o microcontrolador ao computador  
2. Abrir o código-fonte do projeto na IDE  
3. Compilar o código para verificação de erros  
4. Enviar o programa para o microcontrolador
